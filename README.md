# uidai-fraud-detection-
Aadhaar misuse detection  by python EDA

This project analyzes Aadhaar data to detect suspicious usage using Python EDA.

## Tools
- Pandas
- NumPy
- Matplotlib
- Streamlit
- Tabula-py / pdfplumber

## Objective
To identify fake or abnormal Aadhaar usage patterns.

## How to run
1. pip install -r requirements.txt
2. python src/analysis.py
